# Recommendation System Project

This repository contains a complete implementation of a recommendation system based on language model embeddings.  
The project is provided as a runnable Jupyter / Google Colab notebook and focuses on reproducibility, evaluation, and clear result interpretation.

---

## Project Objective
The goal of this project is to:
- Build a recommendation system based on language model embeddings
- Apply appropriate preprocessing and modeling techniques
- Evaluate the performance of the model using relevant metrics
- Analyze the strengths and limitations of the obtained results

This project is intended for academic and educational purposes in the context of machine learning and data science.

## How to Run the Project

### Option 1 — Run on Google Colab (Recommended)
Click the badge below to open and run the notebook directly in Google Colab:

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](
https://colab.research.google.com/github/itsMaherrr/recommendation-project/blob/main/projet_recommandation.ipynb
)

Then select:

Runtime → Run all

---

### Option 2 — Run Locally
1. Clone the repository:
```bash
git clone https://github.com/itsMaherrr/recommendation-project.git
cd recommendation-project
```
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3.Launch the notebook:
```bash
jupyter notebook projet_recommandation.ipynb
```
